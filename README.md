# docker-compose-practice

## Preparation
To test this you need to have:
* [docker](https://docs.docker.com/get-docker/)
* [docker-compose](https://docs.docker.com/compose/install/)

## Testing
```
$ docker-compose build
$ docker-compose up -d
$ curl http://localhost:8000
<!DOCTYPE html>
<html>
<body>

<h1>App1</h1>

<p>APP1 page</p>

</body>
</html>
```
